Tipos de Testes de Software

Testes Manuais

Envolvem a execução direta das funcionalidades da aplicação por uma pessoa, verificando se os comportamentos esperados ocorrem corretamente.

Testes Automatizados

São scripts que executam testes automaticamente, reduzindo esforço repetitivo e aumentando a cobertura de testes.

Testes de API

Focados na validação de endpoints e dados de entrada/saída entre sistemas, normalmente usando ferramentas como Postman.
